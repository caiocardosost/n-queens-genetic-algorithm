# N-Queens Genetic Algorithm

This project was developed during undergraduate studies to solve the N-Queens problem using a Genetic Algorithm implemented in Python.

## Technologies

- Python
- NumPy
- Matplotlib
- Psutil

## Features

- Random population generation
- Fitness evaluation
- Tournament selection
- Crossover and mutation
- Evolutionary optimization
- Fitness visualization
- CPU and RAM usage analysis

## Project Structure

```text
├── src/
└── report/
```

## How to Run

Install dependencies:

```bash
pip install numpy matplotlib psutil
```

Run the project:

```bash
python src/n_queens.py
```

## Objective

The goal of this project is to find valid solutions for the N-Queens problem by minimizing diagonal conflicts between queens through evolutionary optimization techniques.
